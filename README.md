# 🛍️ Zafira – Responsive E-Commerce Website

Zafira is a modern and responsive multi-page e-commerce website built using HTML, CSS, Bootstrap, and JavaScript. This project focuses on creating a clean user interface and smooth user experience similar to real-world online stores.

---

## 🚀 Features

* Fully responsive design (mobile, tablet, desktop)
* Multi-page layout (Home, About, Contact)
* Clean and modern UI
* Smooth animations and transitions
* Scroll-based effects using JavaScript
* Contact form with validation
* Interactive UI components

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* Bootstrap 5
* JavaScript (DOM Manipulation)
